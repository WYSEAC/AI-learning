# AI岗位分析报告

> AI应用工程师 · 全方位能力评估与求职导航系统  
> 数据驱动决策，从画像到入职的完整闭环

---

## 📋 项目简介

本项目是一套面向 **AI应用工程师** 岗位的综合性分析报告系统，涵盖从 **个人能力评估** → **市场数据洞察** → **政策趋势解读** → **学习路径规划** → **简历面试准备** 的完整链路。

所有数据基于智联招聘、BOSS直聘等主流招聘平台的 **2602条** 真实AI岗位数据，结合国家政策文件（国务院、网信办、中国信通院）和权威研究机构报告（IDC、Gartner、清华大学）进行分析。

## 🚀 快速启动

### 前置要求

- **Python 3.7+**（仅数据看板需要启动本地服务）
- **现代浏览器**（Chrome / Edge / Firefox）

### 启动数据看板（模块2 · 核心）

```bash
# 进入看板目录
cd "2.招聘数据分析可视化看板"

# 启动本地HTTP服务
python -m http.server 8001

# 浏览器访问
http://localhost:8001
```

### 数据概况

- **数据来源**：智联招聘 + BOSS直聘

- **覆盖城市**：北京、上海、广州、深圳、杭州、成都、武汉、苏州

- **技能标签**：30项核心技能 + 31项加分技能

## 🔧 数据处理脚本说明

### `process_data.py` 核心功能

1. **薪资解析** — 支持 K/万 单位、范围/固定薪资格式
2. **城市提取** — 从工作地点中提取标准化城市名
3. **经验/学历归一化** — 将原始描述映射为标准分档
4. **技能提取** — 从职位描述中匹配30项核心技能+31项加分技能
5. **岗位名称智能合并** — 将 "AI Agent开发工程师"/"AI Agent工程师" 等相似名称归并
6. **职责/技术要求提取** — 从描述中提取关键职责和技术要点
7. **生成输出** — 产出4个JSON文件供前端使用

### 输出文件

| 文件 | 内容 | 大小 |
|------|------|------|
| `jobs_cleaned.json` | 2602条清洗后的岗位数据 | ~2MB |
| `core_skills.json` | 30项核心技能统计 | ~2KB |
| `plus_skills.json` | 31项加分技能统计 | ~2KB |
| `job_type_details.json` | 688种岗位类型详情摘要 | ~600KB |
